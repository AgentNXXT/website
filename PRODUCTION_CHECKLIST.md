# Production Readiness Checklist

## Pre-Deployment
- [ ] Run `npm install`
- [ ] Run `npm run check`
- [ ] Validate metadata and canonical URLs
- [ ] Verify all internal links
- [ ] Test mobile and desktop layouts
- [ ] Confirm analytics and domain settings

## Deployment
- [ ] Deploy preview
- [ ] Smoke test homepage and key sections
- [ ] Verify robots and sitemap
- [ ] Configure monitoring and uptime checks

## Post-Deployment
- [ ] Submit sitemap to Google Search Console
- [ ] Validate Core Web Vitals
- [ ] Monitor console and runtime errors
